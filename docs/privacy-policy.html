Michi-Navi プライバシーポリシー

最終更新日: 2026年4月6日

Michi-Navi（以下「本アプリ」）は、osprey74（以下「開発者」）が提供する道の駅ナビゲーションアプリです。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

1. 収集する情報

本アプリは以下の情報をデバイス上でのみ使用します。

  • 位置情報: 地図表示、現在地の追従、最寄りの道の駅の表示に使用します。位置情報はデバイス上でリアルタイムに処理され、外部サーバーには送信されません。
  • 写真（フォトアルバム機能）: 道の駅ごとに最大3枚の写真を保存できます。写真はデバイスのローカルストレージにのみ保存され、外部サーバーには送信されません。
  • 設定情報: 地図タイルの選択、お気に入り・到達済みの道の駅リスト等のアプリ設定を、デバイス上のローカルストレージに保存します。

2. 情報の外部送信

本アプリは、ユーザーの個人情報を外部サーバーに送信しません。ただし、以下のサードパーティサービスへのネットワーク通信が発生します。

  • 地図タイル配信サーバー: 地図を表示するために、選択された地図タイルプロバイダー（国土地理院、OpenFreeMap、Google Maps）へタイル画像のリクエストを送信します。これらのリクエストにはユーザーの閲覧範囲の座標情報が含まれます。
  • Wikimedia Commons: 道の駅の施設写真を取得するために、Wikimedia Commonsのサーバーへ画像リクエストを送信する場合があります。

上記の通信はアプリの機能提供に必要なものであり、ユーザーを識別する情報は送信されません。各サードパーティサービスのプライバシーポリシーについては、それぞれの提供元をご確認ください。

3. Google Maps API キー

ユーザーが任意で登録するGoogle Maps APIキーは、デバイスのローカルストレージにのみ保存されます。開発者がこのキーにアクセスすることはありません。

4. データの保存と削除

すべてのデータはデバイス上にのみ保存されます。アプリをアンインストールすると、保存されたすべてのデータ（写真、設定、お気に入りリスト等）が削除されます。

5. 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。13歳未満の子どもから意図的に個人情報を収集することはありません。

6. プライバシーポリシーの変更

本プライバシーポリシーは予告なく変更されることがあります。変更はこのページに掲載された時点で有効となります。

7. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、GitHubリポジトリの Issues までお願いします。

Michi-Navi Privacy Policy

Last updated: April 6, 2026

Michi-Navi (the "App") is a roadside station (Michi-no-Eki) navigation app developed by osprey74 (the "Developer"). This privacy policy explains how information is handled in the App.

1. Information Collected

The App uses the following information on the device only.

  • Location Data: Used for map display, tracking current position, and showing nearby roadside stations. Location data is processed in real-time on the device and is not sent to any external server.
  • Photos (Photo Album Feature): Users can save up to 3 photos per roadside station. Photos are stored only in the device's local storage and are not sent to any external server.
  • Settings: App preferences such as map tile selection, favorite and visited station lists are stored in the device's local storage.

2. External Data Transmission

The App does not transmit personal information to external servers. However, network communication with the following third-party services occurs:

  • Map Tile Servers: To display maps, tile image requests are sent to the selected map tile provider (GSI, OpenFreeMap, or Google Maps). These requests include coordinate information for the viewed area.
  • Wikimedia Commons: Image requests may be sent to Wikimedia Commons servers to retrieve roadside station facility photos.

The above communications are necessary for app functionality. No user-identifying information is transmitted. Please refer to each third-party service's privacy policy for details.

3. Google Maps API Key

The Google Maps API key optionally registered by users is stored only in the device's local storage. The Developer does not have access to this key.

4. Data Storage and Deletion

All data is stored on the device only. Uninstalling the App will delete all stored data, including photos, settings, and favorite lists.

5. Children's Privacy

The App is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

6. Changes to This Policy

This privacy policy may be updated without prior notice. Changes become effective when posted on this page.

7. Contact

For questions about this privacy policy, please use the Issues page on our GitHub repository.
